Pré-requisito: ter uma conta dropbox. Se ainda não tem vá www.dropbox.com e crie uma nova conta.

  • Conecte o MoneyManagerEx para Android com a Dropbox. Seleccione "Link para Dropbox". Se o seu dispositivo tiver instalado a app Dropbox, ele automaticamente a abre e pede-lhe para autorizar o MoneyManagerEx para Android aceder à sua conta Dropbox. Se você não tiver instalado o app Dropbox no seu dispositivo, abra o navegador na página de login do Dropbox. Após o login, você deverá autorizar o MoneyManagerEx para Android para aceder à sua conta.
  • Se você autorizar o MoneyManagerEx para Android, da primeira vez será criada automaticamente pelo Dropbox, a pasta /Apps/MoneyManagerEx Mobile dentro do espaço da Dropbox. A pasta nunca deve ser criada manualmente!

    Eu já uso uma base de dados para a versão desktop
  • Se você já tiver uma base de dados (mmb) da versão de Desktop do MoneyManagerEx, você deve copiar ou mover o ficheiro (s) dentro da pasta /Apps/MoneyManagerEx Mobile (recomendado usar o computador para esta operação).
  • Através das definições de Dropbox do MoneyManagerEx para Android, clique em "ficheiro linkado no Dropbox ". A aplicação abrirá uma nova janela com todos os ficheiros na pasta /Apps/MoneyManagerEx Mobile. Seleccione o ficheiro que deseja conectar-se e volte atrás.
  • Quando voltar ás configurações do Dropbox do MoneyManagerEx para Android, após 10 segundos ele vai começar a descarregar o ficheiro seleccionado, você pode ver a notificação que será aberta no seu dispositivo. Quando o download estiver concluído, o seu dispositivo notifica que o aficheiro está pronto a ser usado, e deve clicar em "Abrir base de dados" para completar a ligação entre o MoneyManagerEx para Android e o Dropbox.

    • Eu nunca usei a versão desktop e quero sincronizar a base de dados no dispositivo
  • Se ainda não tiver um ficheiro de mmb na versão de Desktop do MoneyManagerEx, deverá clicar em "Enviar a base de dados para o Dropbox". Após aproximadamente 10 segundos começará a enviar a base de dados para o Dropbox actualmente em uso. No final, o dispositivo irá notificar a conclusão do envio e na pasta / Apps/MoneyManagerExMobile que irá encontrar um ficheiro chamado data.mmb.

    • Outras definições
  • Se não quer sincronizar automaticamente defina "Intervalo de sincronização" para zero.
  • É importante definir "Envio imediato da base de dados após alterações". Esta configuração permite que o MoneyManagerEx para Android envie a base de dados para o Dropbox imediatamente assim que fizer qualquer alteração.
  • Use "Download da base de dados do Dropbox" ou "Envio para o Dropbox" para forçar a operação de Download ou Upload da base de dados, em caso de mau funcionamento.
  • Quando liga a sua conta Dropbox e o ficheiro na janela principal activa a funcionalidade de sincronização na barra de ferramentas superior.

  • Q. Eu quero sincronizar o MoneyManagerEx para Android com outros serviços na nuvem (Drive, Box.net, Wuala, etc. Como posso fazer?
    A. No MoneyManagerEx para Android, o único serviço de armazenamento na nuvem é o Dropbox. Se deseja usar a base de dados com outros serviços de armazenamento na nuvem, pesquise no Google Play a aplicação que permite que você sincronize com o seu serviço. A base de dados será guardada na memória externa do aparelho e depois aberta com o MoneyManagerEx para Android
  • Q. Após uma sincronização não consigo ver os meus dados. Perdi todos os dados na minha base de dados?
    A. Calma, você não perdeu nada! Em casos raros, quando faz a sincronização, a base de dados não é descarregada correctamente, e não é utilizável para o MoneyManagerEx para Android. Nestes casos, vamos querer forçar um download da base de dados do dropbox. Se mesmo assim não funcionar, nós recomendamos que você apague os ficheiros em /com.money.manager.ex/dropbox/ e force o download do ficheiro.
  • Q. Posso restaurar uma versão anterior da base de dados?
    A. Claro que sim. O Dropbox guarda todas as versões de todos os seus ficheiros. Aqui você pode encontrar o guia do Dropbox em como restaurar ficheiro
