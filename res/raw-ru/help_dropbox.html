MoneyManagerEx для Android: Синхронизация базы данных с MoneyManagerEx для Windows/Linux/Mac используя сервис Dropbox.

Требование: наличие акаунта dropbox. Если у вас до сих пор нет такого акаунта - то вы можите создать его пройдя регистрацию на сайте www.dropbox.com.

  • Подключение MoneyManagerEx для Android к сервису Dropbox. Выберите "Link to Dropbox." Если на вашем устройстве уже установлено приложение Dropbox, оно автоматически откроется и попросит вас автроризовать доступ к сервисам Dropbox для приложения MoneyManagerEx для Android. Если у вас не уставлено приложение Dropbox, Android откроет веб браузер на странице входа в систему Dropbox. После входа в систему, вам необходимо будет авторизовать доступ программы MoneyManagerEx для Android к вашему аккаунту Dropbox.
  • Если вы авторизуете доступ MoneyManagerEx для Android, автоматически при первом использовании будет создан каталог в Dropbox: /Apps/MoneyManagerEx Mobile в вашем разделе Dropbox. Данный каталог нельзя создавать вручную !

    Я уже использую базу ПК версии MoneyManagerEX
  • Если у вас уже есть база программы (файл .mmb) от ПК версии MoneyManagerEx, вы должны скопировать или переместить его (их) в папку /Apps/MoneyManagerEx Mobile (рекомендуется использовать ПК для этой операции).
  • Из раздела настроек MoneyManagerEx для Android - Dropbox, выбирите опцию "Файл Dropbox". Приложение откроет новое окно со списком всех файлов в папке /Apps/MoneyManagerEx Mobile.
