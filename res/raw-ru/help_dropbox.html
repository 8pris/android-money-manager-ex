MoneyManagerEx для Android: Синхронизация базы данных с MoneyManagerEx для Windows/Linux/Mac используя сервис Dropbox.

Требование: наличие акаунта dropbox. Если у вас до сих пор нет такого акаунта - то вы можите создать его пройдя регистрацию на сайте www.dropbox.com.

  • Подключите MoneyManagerEx для Android к сервису Dropbox. Выберите "Соединиться с Dropbox." Если на вашем устройстве установлено официальное приложение сервиса Dropbox, оно автоматически откроется и попросит вас авторизовать доступ к сервису Dropbox приложению MoneyManagerEx для Android. Если у вас на устройстве не установлено офицальное приложение Dropbox, система Android откроет веб браузер на странице входа в сервис Dropbox. После входа на сайт сервиса Dropbox под вашей учетной записью, необходимо будет авторизовать доступ программы MoneyManagerEx для Android к вашему сервису Dropbox.
  • После авторизации MoneyManagerEx для Android, при первом использовании сервиса, следующая папка будет создана автоматически для вас: /Приложении/MoneyManagerEx Mobile для вашего аккаунта Dropbox. Этот каталог нельза создавать вручную !

    Я уже использую базу ПК версии MoneyManagerEX
  • Если у вас уже есть файл с базой данных (mmb) от программы MoneyManagerEx версии для ПК, необходимо скопироватьили переместить этот файл(ы) в каталог /Приложении/MoneyManagerEx Mobile сервиса Dropbox (рекомендуется использовать ПК для этой цели).
  • Из раздела настроек сервиса Dropbox в программе MoneyManagerEx для Android выберите опцию "Файл Dropbox". Приложение откроет новое окно со списком всех файлов в папке /Приложении/MoneyManagerEx Mobile. Выберите требуемый файл для синхронизации.
  • После этого, когда вы вернетесь в настройки Dropbox программы MoneyManagerEx для Android, после примерно 10 секунд программаначнет загрузку выбранного файла, будет показано стандартное уведомление системы Android об окончании загрузки. После того как загрузка завершена, ваше устройство проинформирует вас что файл загружен и готов к использованию в разделе стандартных уведомлений Android, обычно в верхней части экрана и вы должны выбрать опцию "Открыть базу данных" для того, чтобы завершить установления связи между сервисом Dropbox и программой MoneyManagerEx для Android.

    • Я никогда не использовал ПК версию MoneyManagerEx и я хочу синхронизировать базу на мобильном устройстве
  • Если у вас нет файла базы данных mmb от ПК векрсии программы MoneyManagerEx , вы должны выбрать опцию "Загрузка в Dropbox". После примерно 10 секунд программа нечнет загрузку текущей базы данных на сервис Dropbox. По окончании загрузки система проинформирует вас стандартным уведомлением Android, и в папке /Приложении/MoneyManagerEx Mobile вы найдете файл data.mmb.

    • Прочие настройки
  • Если вы не хотите использовать автоматическую синхронизацию, установите значение в ноль для опции "Интервал синхронизации".
  • Важно установить опцию "Загрузка сразу после изменения". Данная настройка позволяет программе MoneyManagerEx для Android загружать измененную базу данных на сервис Dropbox сразу после внесения изменений на мобильном устройстве, для этого требуется постоянный доступ в интернет.
  • Используйте опции "Загрузка из Dropbox" или "Загрузка в Dropbox" для принудительной загрузки базы из или в Dropbox.
  • После установления связи с сервисом Dropbox активируется опция синхронизации в основном меню.

    • FAQ
  • В. Я не могу подключиться к сервису Dropbox или синхронизация не работает ?
    О. .
  • В. Я хочу синхронизировать данные приложения MoneyManagerEx для Android с другими облачными сервисами (Drive, Box.net, Wuala, итд. ..). Как мне это сделать ?
    О. В программе MoneyManagerEx для Android используется встроенный метод синхронизации только с сервисом Dropbox. Если вы хотите синхронизировать файл базы данных с другим онлайн сервисом, необходимо найти приложение на Google Play которое позволит проводить такого рода синхронизацию. База данных программы MoneyManager для Android будет в таком случае сохранена на внешнюю карту памяти устройства и вам будет необходимо открыть ее после синхронизации в программе MoneyManagerEx для Android
  • В. После синхронизации я не вижу своих данных. Означает ли это что данные в базе потеряны ?
    О. Ничего не потеряно ! В редких случаях, когда синхронизируемая база данных была загружена не полностью она не может быть использована в программе MoneyManagerEx для Android. В таких случаях, мы рекомендуем опцию принудительной загрузки базы данных в или из Dropbox в зависимости от типа предыдущей операции в результате которой произошла ошибка передачи файла. Если ошибка произошла при загрузке обновленной базы из сервиса Dropbox возможное решение будет создание резервной копии файлов из каталога /com.money.manager.ex/dropbox/ , затем очистка этого каталога и после этого попытка повторной загрузки из сервиса Dropbox.
  • В. Возможно ли восстановить предыдущую версию базы ?
    О. Конечно это возможно. Dropbox сохраняет все версии ваших файлов. Здесь вы найдете руководство сервиса Dropbox о том как восстановить файлы
